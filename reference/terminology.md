# Dementia Research Terminology (專業術語對照表)

> 為了保持學術研究的嚴謹性，本手冊中的專業術語均保留英文原名，並在此提供詳細的中文解析與邏輯背景。

| 術語 (Term) | 中文解析 (Explanation) | 所屬維度 (Axis) | 核心邏輯 (Core Logic) |
| :--- | :--- | :--- | :--- |
| **Aβ (Amyloid-beta)** | 類澱粉樣蛋白 | Axis A | 在神經元外沉積形成「斑塊 (Plaques)」，是 AD 的主要病理特徵之一。 |
| **Tau Protein** | Tau 蛋白 | Axis A | 在神經元內過度磷酸化並形成「纏結 (Tangles)」，直接導致神經元死亡。 |
| **Proteostasis** | 蛋白質穩態 | Axis A | 維持蛋白質合成、折疊、運輸與降解平衡的機制，其崩潰是退化性疾病的源頭。 |
| **Primary Nucleation** | 初級成核 | Axis A | 單體蛋白自發結合形成第一個「種子 (Seed)」的過程，是聚合動力學的速控步驟。 |
| **BBB (Blood-Brain Barrier)** | 血腦屏障 | Axis E/B/C | 由內皮細胞組成的物理與生化屏障，保護大腦免受血液中污染與病原體侵害。 |
| **Glymphatic System** | 類淋巴系統 | Axis A/C | 大腦的廢物清除系統，主要在睡眠中運作，負責排除 Aβ 等代謝廢物。 |
| **Microglia** | 微膠質細胞 | Axis B | 大腦的駐留免疫細胞，負責監視環境、吞噬斑塊與引發炎症。 |
| **DAM (Disease-Associated Microglia)** | 疾病相關微膠質細胞 | Axis B | 一種特定的免疫狀態，在病理環境下被激活，具有保護與毒性的雙重性質。 |
| **Neuro-immune Niche** | 神經免疫生態位 | Axis B | 大腦內特定空間區域（如斑塊周圍），聚集了免疫細胞、神經元與病理蛋白的交互中心。 |
| **Gut-Brain Axis** | 腸腦軸 | Axis C | 腸道微生物及其代謝物透過迷走神經或循環系統與大腦進行雙向通訊。 |
| **Irisin (鳶尾素)** | 肌肉激素 | Axis C | 運動時由肌肉分泌的一種激素，能穿透 BBB 並誘導大腦 BDNF 分泌。 |
| **BDNF** | 腦源性神經營養因子 | Axis C/A | 促進神經元存活、生長與突觸可塑性的關鍵蛋白質。 |
| **Spatial Transcriptomics** | 空間轉錄組學 | Axis D | 在保持細胞空間位置信息的同時，測量其基因表達，實現「原位」解析。 |
| **scRNA-seq** | 單細胞 RNA 測序 | Axis D/B | 在單個細胞維度解析基因表達，揭示細胞群體的異質性 (Heterogeneity)。 |
| **PINNs** | 物理信息神經網絡 | Axis F | 將生物物理規律（如擴散方程）嵌入神經網絡，用於預測數據稀疏的生物過程。 |
| **pLMs** | 蛋白質語言模型 | Axis F | 利用 Transformer 學習蛋白質序列的「語法」，預測蛋白結構、功能與聚集傾向。 |
| **MNPs (Micro/Nanoplastics)** | 微塑料與奈米塑料 | Axis E | 環境中直徑 <5mm (Micro) 及 <1μm (Nano) 的塑料碎片，能穿透 BBB 進入大腦。 |
| **Protein Corona** | 生物冠 | Axis E | 進入生物體的塑料顆粒表面吸附的一層血漿蛋白，決定了其生物學歸宿與穿透力。 |
| **GSDMD (Gasdermin D)** | 細胞焦亡蛋白 | Axis B/E | 被激活後在細胞膜上形成孔道，引發細胞焦亡並導致 BBB 完整性破裂。 |
| **GENUS** | 感覺神經夾帶激發 | Axis A/B | 通過 40Hz 的光/聲刺激誘導大腦產生 Gamma 波，促進 Aβ 清除與免疫調節。 |
| **VIP (Vasoactive Intestinal Peptide)** | 血管活性腸肽 | Axis A/C | 40Hz 刺激下釋放的關鍵神經肽，能擴張血管並增強動脈搏動，驅動類淋巴系統。 |
| **Ferroptosis** | 鐵死亡 | Axis E/B | 一種依賴於鐵的程序性細胞死亡形式，常由重金屬暴露引發的氧化應激觸發。 |
| **Histone Acetylation** | 組蛋白乙酰化 | Axis E/C | 一種表觀遺傳修飾，農藥暴露可透過此機制長期「啟動」免疫細胞。 |
| **Noise-induced Arousal** | 噪音誘導覺醒 | Axis E/A | 噪音導致藍斑核釋放去甲腎上腺素，物理性關閉類淋巴清除系統。 |
| **Digital Twin** | 數位孿生 | Axis D/F | 建立患者大腦的虛擬模型，整合多組學數據以模擬病程發展與藥物反應。 |
| **AI Surrogate Brain** | 人工智能替代大腦 | Axis F | 利用數據驅動方法精確預測全腦動態行為，作為數位孿生的核心計算引擎。 |
| **Neural ODEs** | 神經常微分方程 | Axis F | 一種深度學習架構，用於建模隨時間連續變化的生物系統（如蛋白聚集或腦網絡演化）。 |
| **Epigenetic Mapping** | 表觀遺傳圖譜 | Axis C | 記錄基因表達受環境影響（而非基因序列改變）的修飾狀態（如 DNA 甲基化）。 |
| **scMulti-omics** | 單細胞多組學 | Axis D | 同時在單個細胞中測量轉錄組、蛋白質組、表觀組等多個維度的數據。 |
| **p-tau217** | 磷酸化 Tau 217 | Axis D/A | 血液中特定的 Tau 蛋白亞型，是目前最精確的失智症早期診斷生物標誌物。 |
| **Bmal1 / Clock** | 晝夜節律基因 | Axis G | 調控生物鐘的核心基因，影響微膠質細胞的吞噬能力與類淋巴系統的脈衝式清除。 |
| **Slow-wave Sleep** | 慢波睡眠 | Axis G | 深度睡眠階段，是類淋巴清除系統運行的黃金窗口，對 p-tau 的清除至關重要。 |
| **Pericytes** | 周細胞 | Axis H | 包繞在毛細血管外的細胞，控制血流速度與 BBB 完整性，其丟失是血管性退化的起始。 |
| **Neurovascular Coupling** | 神經血管耦合 | Axis H | 神經元活動與局部血流量之間的精確匹配機制，AD 早期該機制會發生失靈。 |
| **Fibrinogen** | 纖維蛋白原 | Axis H/B | 血液中的凝血蛋白，一旦滲透進入大腦實質，會直接誘導微膠質細胞產生促炎反應。 |
| **Metabolic Reprogramming** | 代謝重編程 | Axis I | 神經元在應激下從正常的葡萄糖代謝轉向脂肪酸氧化等損耗性路徑的過程。 |
| **Insulin Resistance** | 胰島素抵抗 | Axis I | 細胞對胰島素信號不敏感，導致大腦「血糖飢餓」，觸發 Tau 蛋白病理變化。 |
| **Neuronal Glycogen** | 神經元糖原 | Axis I | 正常情況下極少，在 AD 早期異常積累，被視為代謝紊亂與蛋白聚集的預警信號。 |
| **OCT-A** | 光學相干斷層掃描血管造影 | Axis J | 非侵入性影像技術，可精確測量視網膜微血管密度，是大腦微循環狀態的「窗口」。 |
| **RNFL** | 視網膜神經纖維層 | Axis J | 視網膜中的神經組織層，其變薄往往早於認知症狀出現，是早期診斷的關鍵指標。 |
| **Cellular Senescence** | 細胞衰老 | Axis K | 細胞喪失分裂能力但仍維持代謝活性的狀態，是慢性炎症與神經退行的底層驅動力。 |
| **SASP** | 衰老相關分泌表型 | Axis K | 衰老細胞分泌的促炎細胞因子、趨化因子與蛋白酶組合，會「感染」周圍健康細胞。 |
| **Senolytics** | 衰老細胞清除藥物 | Axis K | 一類能特異性誘導衰老細胞死亡的藥物（如 D+Q），旨在重塑大腦的微環境。 |
| **Epigenetic Clock** | 表觀遺傳時鐘 | Axis K | 基於 DNA 甲基化水平預測生物年齡的方法，可用於評估大腦退化程度與干預效果。 |
| **LDAM** | 脂質積聚微膠質細胞 | Axis B/I | 含有大量脂滴的免疫細胞亞群，喪失了正常的吞噬功能，轉而釋放毒性脂質。 |
