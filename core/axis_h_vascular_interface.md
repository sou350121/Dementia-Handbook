# Axis H: Vascular-Neurovascular Interface (神经血管界面与血流动力学)

## 1. 核心逻辑 (Core Logic)
大脑是代谢极其旺盛的器官，神经退行性变往往始于微血管系统的崩溃。神经元与血管之间的“神经血管耦合 (Neurovascular Coupling)”受损，是早期失智症的核心特征，甚至早于 Aβ 斑块的出现。

## 3. 关键机制 (Key Mechanisms)

### A. 周细胞 (Pericyte) 的功能障碍
- **机制**：周细胞负责控制微血管的直径与血流速度。在 AD 早期，周细胞丢失导致毛细血管发生持续性收缩（毛细血管停滞），减少脑血流量。
- **2025 线索**：研究发现 ApoE4 携带者的周细胞退化速度显著加快，这解释了其高风险的血管因素。

### B. 血脑屏障 (BBB) 的“隐形”渗漏
- **机制**：血管内皮细胞之间的紧密连接破裂，导致血液中的纤维蛋白原 (Fibrinogen) 等促炎因子进入大脑。
- **2024 情报**：纤维蛋白原进入实质后，会直接诱导微胶质细胞产生 ROS (Axis E)，形成恶性循环。

---

## 4. 跨轴线交互 (Cross-Axis Interaction)
- **Axis H x Axis E**：微塑料 (MNPs) 随血液循环停留在受损的微血管中，诱发“微血栓”，进一步加剧 Axis H 的灌注不足。
- **Axis H x Axis D**：利用高分辨率 7T-MRI 影像技术监测“血管源性”生物标志物，实现早期预警。
