# Axis H: Vascular-Neurovascular Interface (神經血管界面與血流動力學)

## 1. 核心邏輯 (Core Logic)
大腦是代謝極其旺盛的器官，神經退行性變往往始於微血管系統的崩潰。神經元與血管之間的「神經血管耦合 (Neurovascular Coupling)」受損，是早期失智症的核心特徵，甚至早於 Aβ 斑塊的出現。

## 3. 關鍵機制 (Key Mechanisms)

### A. 周細胞 (Pericyte) 的功能障礙
- **機制**：周細胞負責控制微血管的直徑與血流速度。在 AD 早期，周細胞丟失導致毛細血管發生持續性收縮（毛細血管停滯），減少腦血流量。
- **2025 線索**：研究發現 ApoE4 攜帶者的周細胞退化速度顯著加快，這解釋了其高風險的血管因素。

### B. 血腦屏障 (BBB) 的「隱形」滲漏
- **機制**：血管內皮細胞之間的緊密連接破裂，導致血液中的纖維蛋白原 (Fibrinogen) 等促炎因子進入大腦。
- **2024 情報**：纖維蛋白原進入實質後，會直接誘導微膠質細胞產生 ROS (Axis E)，形成惡性循環。

---

## 4. 跨軸線交互 (Cross-Axis Interaction)
- **Axis H x Axis E**：微塑料 (MNPs) 隨血液循環停留在受損的微血管中，誘發「微血栓」，進一步加劇 Axis H 的灌注不足。
- **Axis H x Axis D**：利用高解析度 7T-MRI 影像技術監測「血管源性」生物標誌物，實現早期預警。
